# Staff Performance Portal V2

Production static frontend for GitHub Pages with Cloudflare Worker API integration.

## Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages hosting
- Cloudflare Worker API
- Google Apps Script and Google Sheets backend behind the Worker

## API

All calls are sent through:

`https://staff-portal-api-v2.mdrobiulislam.workers.dev`

The frontend never calls the Apps Script URL directly.
